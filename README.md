# Sprout Fridge is a family-focused smart kitchen app for managing fridge inventory, expiration alerts, recipe suggestions, grocery planning, and household food preferences. It includes a React frontend, FastAPI backend, MongoDB persistence, simulated fridge scanning, freshness tracking, recipe matching, and auto-replenishment workflows.
